# Procedimento de Instalação

*Clone o repositório:*
```bash
git clone https://github.com/Lunes-platform/install_node.git
```

*Execute o instalador:*
```bash
install_node/install.sh
```

## Prereqs:
1 - estar logado como root ou sudo bash
